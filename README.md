widenbot-flipit
----

An (work-in-progress) flipit plugin for widenbot.

# Commands

```
!flipit
```

# Installation

1. Add dependency to your bot project:

```
npm install --save widenbot-flipit
```

2. Enable in config `plugins`:

```
module.exports = {
    plugins : {
        // ...
        flipit: {}
    }
}
```
